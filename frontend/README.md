# Frontend (React + Vite)

Minimal React shell for Candid with a chat input and response panel.

## Run locally

1. `cd frontend`
2. `npm install`
3. `npm run dev`

By default, the app calls backend `http://localhost:8000/api/v1`.
Override with:

- `VITE_API_BASE_URL`

Example:

- `VITE_API_BASE_URL=http://localhost:8000/api/v1 npm run dev`
